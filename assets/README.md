# Assets

Place your application logo file here:

- Preferred file name: `logo.svg`
- Alternative file name: `logo.png`
- Recommended format: PNG with transparent background
- Recommended size: `512x512` (or larger square image)

The startup screen tries to load `assets/logo.svg` first, then `assets/logo.png`.
If the file is missing or invalid, the app uses an internal fallback logo.
